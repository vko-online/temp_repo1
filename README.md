# Activity mobile app

Install packages

    ❯ yarn start

Install Expo CLI

    ❯ npm install -g exp

Start simulator 

    ❯ open /Applications/Xcode.app/Contents/Developer/Applications/Simulator.app/

Start app

    ❯ exp start
